# Análise de Dados do Brasileirão

Projeto desenvolvido na faculdade para a disciplina de **Programação Orientada a Objetos (POO)**.
O objetivo foi criar um sistema em Java que lê o histórico de jogos do campeonato (2003-2023) e gera estatísticas automáticas.

# Funcionalidades
- Leitura de arquivos de dados (CSV).
- Processamento de informações de Times e Estados.
- Geração de Rankings (Quem fez mais gols, pontuação e medalhas).
